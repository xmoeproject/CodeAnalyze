Macro that returns a pointer to the data structure that contains a specified field of that data structure. \hyperlink{namespace_this}{This} is a lightweight method to hide information by placing a public data structure inside a larger private data structure and using a pointer to the public data structure to retrieve a pointer to the private data structure.

\hyperlink{namespace_this}{This} function computes the offset, in bytes, of field specified by Field from the beginning of the data structure specified by T\+Y\+PE. \hyperlink{namespace_this}{This} offset is subtracted from Record, and is used to return a pointer to a data structure of the type specified by T\+Y\+PE. If the data type specified by T\+Y\+PE does not contain the field specified by Field, then the module will not compile.

{\em Record} & Pointer to the field specified by Field within a data structure of type T\+Y\+PE. \\
\hline
{\em T\+Y\+PE} & The name of the data structure type to return. \hyperlink{namespace_this}{This} data structure must contain the field specified by Field. \\
\hline
{\em Field} & The name of the field in the data structure specified by T\+Y\+PE to which Record points.\\
\hline
\end{DoxyParams}
