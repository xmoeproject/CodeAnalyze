Gets a \hyperlink{struct_line}{Line} from a Text \hyperlink{class_file}{File}, but no more than ul\+Limit charachters. The line will be N\+U\+LL terminated. 

The behaviour of this function is undefined when called on a binary file. \hyperlink{class_it}{It} should just read in ul\+Limit bytes of binary, and Z\+E\+RO terminate the line.

\hyperlink{namespace_this}{This} function works for both U\+N\+IX line feeds, and Windows C\+R\+LF type files.

\end{DoxyParams}
