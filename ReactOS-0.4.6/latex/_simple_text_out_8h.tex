\subsection{详细描述}
Simple Text Out protocol from the U\+E\+FI 2.\+0 specification.

Abstraction of a very simple text based output device like V\+GA text mode or a serial terminal. The Simple Text Out protocol instance can represent a single hardware device or a virtual device that is an aggregation of multiple physical devices.
