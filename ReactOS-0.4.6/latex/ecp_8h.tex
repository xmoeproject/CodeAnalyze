Multiplication by an integer\+: R = m $\ast$ P (Not thread-\/safe to use same group in multiple threads) 

\begin{DoxyNote}{注解}
In order to prevent timing attacks, this function executes the exact same sequence of (base field) operations for any valid m. \hyperlink{class_it}{It} avoids any if-\/branch or array index depending on the value of m.

If f\+\_\+rng is not N\+U\+LL, it is used to randomize intermediate results in order to prevent potential timing attacks targeting these results. \hyperlink{class_it}{It} is recommended to always provide a non-\/\+N\+U\+LL f\+\_\+rng (the overhead is negligible).
\end{DoxyNote}
