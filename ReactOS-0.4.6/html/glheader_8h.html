宏定义说明

◆ GL_SHADER_PROGRAM_MESA

#define GL_SHADER_PROGRAM_MESA   0x9999

Internal token to represent a GLSL shader program (a collection of one or more shaders that get linked together). Note that GLSL shaders and shader programs share one name space (one hash table) so we need a value that's different from any of the GL_VERTEX/FRAGMENT/GEOMETRY_PROGRAM tokens.
