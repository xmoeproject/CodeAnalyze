27 /* The above definition of TAGID_ROOT is used in winapi and can be found
28  * on msdn it but doesn't make sense, especially internally, because
29  * TAGID represents offset into database data and there is a header at
30  * offset 0, NOT a tag. Therefore, this definition should be used internally
31  * to represent first valid TAGID. Header size is 12 bytes. */
