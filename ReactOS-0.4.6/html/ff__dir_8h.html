FF_ERROR FF_FindNext ( FF_IOMAN *  pIoman,
FF_DIRENT *  pDirent 
)

Get's the next Entry based on the data recorded in the FF_DIRENT object.

All values recorded in pDirent must be preserved to and between calls to FF_FindNext(). Please see

参见
FF_FindFirst() for find initialisation.
参数
pIomanFF_IOMAN object that was created by FF_CreateIOMAN().
pDirentFF_DIRENT object to store the entry information. (As initialised by FF_FindFirst()).
返回
FF_ERR_DEVICE_DRIVER_FAILED is device access failed.
